# NumberGuessGame

## 概要
Javaで作成した数当てゲームです。  
ランダムに生成された数字を予想し、正解するまで繰り返します。

## 機能
- 1〜99のランダムな数字を生成
- プレイヤーが数字を入力
- 「大きい」「小さい」のヒント表示
- 正解まで繰り返し
- 正解までの回数を表示

## 使用技術
- Java

## 実行方法
1. プログラムを実行
2. 1〜99の数字を入力
3. ヒントをもとに正解を目指す

## 工夫した点
- do-while文を使い、正解するまで繰り返す処理を実装
- if文を使ってヒント（大小）を表示
- 回数カウント機能を追加し、ゲーム性を向上
